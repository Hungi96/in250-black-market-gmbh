# Sicherheitsrichtlinie

## Meldung von Sicherheitsproblemen

Die Sicherheit dieses Projekts ist uns wichtig. Solltest du eine Sicherheitslücke oder ein sicherheitsrelevantes Problem entdecken, bitten wir dich, uns darüber zu informieren.

Am einfachsten kannst du dies tun, indem du ein neues GitHub Issue in diesem Repository erstellst. Beschreibe das Problem möglichst genau und füge, falls vorhanden, Schritte zur Reproduktion oder weitere hilfreiche Informationen hinzu.

Wir werden die Meldung prüfen und uns bemühen, das Problem zeitnah zu beheben.

Vielen Dank für deine Unterstützung bei der Verbesserung der Sicherheit dieses Projekts.
