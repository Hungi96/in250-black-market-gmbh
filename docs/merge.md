# Merge Konflikt A+B

Diese Datei entstand im Rahmen der IN250 Praxisaufgabe zum Thema Merge-Konflikte.

Der Konflikt wurde absichtlich erzeugt, indem zwei Branches (`feat/MergeA` und `feat/MergeB`)
dieselbe Datei verändert haben. Beim Zusammenführen der Branches entstand ein Konflikt,
der anschliessend manuell gelöst wurde.

Als Lösung wurde der Titel dieser Datei auf **Merge Konflikt A+B** angepasst.
