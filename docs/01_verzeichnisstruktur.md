# Verzeichnisstruktur

## docs
Enthält Dokumentationen des Projekts im Markdown Format.

## website
Enthält die HTML Website der Black Market Software GmbH.

## images
Enthält Bilder, die aus dem Archiv bilder.zip extrahiert wurden.

## sources
Enthält Quellcode aus dem Archiv code.zip.

## documents
Enthält ältere Dokumentationen im Word Format aus dokumentation.zip.

## secrets
Enthält sensible Konfigurationsdateien und geheime Daten des Projekts.
