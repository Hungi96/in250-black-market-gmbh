# Black Market Software GmbH

Willkommen im Repository der **Black Market Software GmbH**.

Dieses Repository wurde im Rahmen des Moduls **IN250 – Development Environment** erstellt.  
Im Verlauf des Projekts wurde ein bestehendes Repository analysiert, neu strukturiert und dokumentiert, um eine übersichtliche und nachvollziehbare Projektstruktur zu schaffen.

---

## Projektbeschreibung

Die Black Market Software GmbH entwickelt moderne Softwarelösungen mit Fokus auf Struktur, Qualität und Wartbarkeit.

Im Rahmen dieses Projekts wurden archivierte Dateien analysiert, extrahiert und in eine saubere Verzeichnisstruktur überführt. Zusätzlich wurde eine Dokumentation erstellt, um zukünftigen Entwicklern einen schnellen Einstieg in das Projekt zu ermöglichen.
